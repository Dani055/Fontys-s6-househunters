The 'admin' , 'config' and 'local' databases can be ignored. They are part of MongoDB's internal setup. As for the other databases:
The 'hh-users' database is managed by the authentication microservice and contains the application's users. It contains a single schema called 'users'

The 'hh-listings' database is managed by the listing microservice and contains 2 schemas - listings and comments
